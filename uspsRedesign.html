USPS Scheduler Form Redesign


Background

Recently, I had to use the USPS Form the schedule an appointment and found numerous UX/UI issues that I have sought to address with this form redesign. The most glaring issue with this form is that it a multi-step form designed as a single-step one. This means that a user can not complete a step without completing one(s) before it, even though it may seem like it. This meant separating out each step and introducing a step-tracker at the top.


There are 2 versions for the first step:

  • Version 1 uses a "dropdown" for selecting the number of adults and the number of minors.
  • Version 2 uses radio buttons for both the options. The use of radio buttons makes so that it is not as easy to scale as Version 1, however, the time to complete the step and hence the form would be lesser in comparison.

Each version has pre-selected for the fields to speed up the process.



For the second step, one of the search by options could be pre-selected based on which is more frequently selected by users. For the purposes of this demonstration have assumed that that option is location.
